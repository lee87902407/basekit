# 功能示例文档

本目录用于存放按主模块拆分的示例文档。

## 约定

当新增一个主模块时，需要新增对应文档：

- `docs/examples/<module>.md`

每篇文档建议包含：

1. 模块用途。
2. 典型使用场景。
3. 最小可运行示例。
4. 接入注意事项。
5. 与其他模块的关系（如有）。

## 当前状态

当前已存在正式主模块示例文档：

- [`mempool.md`](./mempool.md)：`mempool` 模块的设计背景、使用场景、最小示例与注意事项。
- [`log.md`](./log.md)：`log` 模块的初始化方式、输出模式、最小示例与接入注意事项。

后续新增模块后，请同时：

1. 在本目录添加对应示例文档。
2. 在 `README.md`、`README.zh-CN.md`、`README.en.md` 中补充跳转入口。
3. 在 `examples/` 中补充相应示例代码。
